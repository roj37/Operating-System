Operating system Function
Memory Management- refers to the OS managing primary and main memory. Main memory is a large array of words or 
bytes where each word or byte has its own address. Main memory provides a fast storage that can be directly accessed
by the CPU. For a program to be executed it must be in Main memory.

Another OS Function
Processor Management- In a multiprogramming environment the OS decides which process gets the processor when and for how much time. This function is called process scheduling. It also allocates The CPU to a specific process. De-allocates the CPU
when it is no longer needed. Keeps track of the processor and its status.
